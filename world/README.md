# World Economic Data Extraction
This folder contains Python scripts for automated data retrieval from major international economic institutions.
## 🏛️ Data Sources & Coverage
### [BIS] Bank for International Settlements (*Upcoming*)
- **Database:** Central bank policy rates.
- **Upcoming Script:**
  - `BIS.ipynb`: Fetches Policy Rates data for all countries available.
### [IMF] International Monetary Fund (*Upcoming*)
- **Database:** Consumer Price Index.
- **Upcoming Script:**
  - `IMF.ipynb`: Fetches Consumer Price Index (Year-Over-Year % Change) data for all countries available.
### [OECD] Organisation for Economic Co-operation and Development (*Upcoming*)
- **Database:** Economic Outlook.
- **Upcoming Script:**
  - `OECD.ipynb`: Fetches Output Gap (as a percentage of GDP) data for all countries members of OECD.
### [WB] World Bank
- **Database:** International Debt Statistics.
- **Script:**
  - `World_Bank.ipynb`: Fetches External debt stocks (as a percentage of GNI) data for a selection of Latin American countries.
