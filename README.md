# econ_apis

## Economic Data APIs with Python

Contact: marco.virul@pucp.edu.pe

This repository contains examples for downloading economic data from public Application Programming Interfaces (APIs) of institutions using Python. Each script demonstrates the data retrieval process.

In some cases, an API key is required.

### Contents

**United States**
- Bureau of Economic Analysis (BEA). *Publication date: March 13*
- [Bureau of Labor Statistics (BLS)](https://github.com/marcovirulucas/econ_apis/blob/main/united_states/BLS_inflation.ipynb)
- Fiscal Data. *Publication date: March 11*
- FRED. *Publication date: March 31*

**Peru**
- Banco Central de Reserva del Perú (BCRP). *Publication date: March 20*

**World**
- International Monetary Fund (IMF). *Publication date: April 7*
- Organisation for Economic Co-operation and Development (OECD). *Publication date: March 27*
- [World Bank](https://github.com/marcovirulucas/econ_apis/blob/main/world/World_Bank.ipynb)
